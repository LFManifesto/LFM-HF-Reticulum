# Contributing to ReticulumHF

Thank you for your interest in contributing. This project documents working configurations for Reticulum over HF radio. Contributions that expand hardware support, improve documentation, or fix errors are welcome.

## How to Contribute

### Reporting Issues

- Use GitHub Issues for bugs, errors in documentation, or feature requests
- Include your hardware configuration and software versions
- Provide steps to reproduce any problems

### Pull Requests

**All contributions must be tested on actual hardware before submission.**

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/new-radio-support`)
3. Test your changes on real equipment
4. Commit with clear, descriptive messages
5. Push to your fork and open a Pull Request

### What We're Looking For

**High Priority:**
- Tested configurations for additional radios (IC-7300, FT-891, etc.)
- Troubleshooting guides for common issues
- Performance optimization tips
- Regional regulatory guidance

**Also Welcome:**
- Typo and clarity fixes
- Improved diagrams or visuals
- Translation to other languages

### Standards

- **Accuracy over speed** - Only submit what you've personally verified
- **Specificity** - Include exact hardware models, software versions, and settings
- **Completeness** - Document all steps, even "obvious" ones
- **Brevity** - Keep instructions clear and concise

## Code of Conduct

- Be respectful and constructive
- Focus on technical merit
- No classified, ITAR-controlled, or illegal content
