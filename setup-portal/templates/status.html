Connect via SSH

Open a terminal and run:

ssh pi@{{ system_info.ip_address or 'reticulumhf.local' }}

Password: reticulumhf

Connect with Sideband or Columba

Step 1: Install app on your phone

  • Sideband (Android/Linux/macOS/Windows)
  • Columba (Android - lighter weight)

Step 2: Connect phone to WiFi

Network: ReticulumHF

Step 3: Configure Sideband TCP Interface

In Sideband: Settings → Connectivity → Add Interface → TCP Client

TCP Host{{ system_info.ip_address or '192.168.4.1' }}
TCP Port4242
IFAC Network Name(not set)
IFAC Passphrase(not set)

Step 4: Start messaging!

Messages route: Phone → Gateway → Radio → HF

Advanced: SSH Access

For terminal access and diagnostics:

ssh pi@{{ system_info.ip_address or '192.168.4.1' }}

Diagnostic Commands (via SSH)

rnstatus Check Reticulum network
journalctl -u freedvtnc2 -f View HF modem logs
aplay -l List audio devices

Adjusting Audio Levels

If your signal is too loud or quiet, adjust via SSH:

amixer -c 1 sset 'Speaker' 64% Output level
amixer -c 1 sset 'Mic' 75% Input level

Replace -c 1 with your audio card number from aplay -l
